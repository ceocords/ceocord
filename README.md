# CeoCord

![](https://img.shields.io/github/package-json/v/ceocord/ceocords?style=for-the-badge&logo=github&logoColor=d3869b&label=&color=1d2021&labelColor=282828)

The cutest Discord client mod


## Features

-   Easy to install
-   [100+ built in plugins](https://ceocord.dev/plugins)
-   Fairly lightweight despite the many inbuilt plugins
-   Excellent Browser Support: Run CeoCord in your Browser via extension or UserScript
-   Works on any Discord branch: Stable, Canary or PTB all work
-   Custom CSS and Themes: Inbuilt css editor with support to import any css files (including BetterDiscord themes)
-   Privacy friendly: blocks Discord analytics & crash reporting out of the box and has no telemetry
-   Maintained very actively, broken plugins are usually fixed within 12 hours
-   Settings sync: Keep your plugins and their settings synchronised between devices / apps (optional)
